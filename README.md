## 第三方依赖介绍
- chalk 终端字体颜色
- log-symbols：在终端上显示√或×等图标
- ora：终端显示下载中的动画、
- fs-extra：fs模块扩展版
- inquirer:通用的命令行用户界面集合，用于交互
- commander：解析命令和参数，用于处理用户输入的命令
- shelljs：用于在 Node.js 中执行 shell 命令的模块
- simpleGit: 用于执行git命令
